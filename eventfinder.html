Overview

Event Finder is a serverless, progressive web application (PWA) built using React. The app is created based on a test-driven development (TDD) technique, and fetches calender events from the Google Calendar API.

Purpose

The Event Finder app was built as part of my web development coursework through CareerFoundry. The goal of the project was to build a PWA that used serverless functions and worked offline. My additional goal was to feature a aesthetically pleasing user interface.
