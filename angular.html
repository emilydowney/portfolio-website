Overview

Angular client-side for a movie application called FaveFlix. Built using existing server-side code (REST API and database). Users are ables to create accounts and view some of their favorite movies. This project was generated with Angular CLI version 12.2.6.

Purpose

The FaveFlix(Angular Client) app was built as part of my web development coursework through CareerFoundry. The goal of the project was to learn more about Angular structure and the use of TypeScript. My additional goal was to implement functionality unique to Angular.
