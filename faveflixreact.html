Overview

Client-side for a movie application(FaveFlix) that serves several interface views and was built using the React library. FaveFlix is a single page application with React-Bootstrap incorporated for some of the design elements. Parcel is used as the build tool.

Purpose

The FaveFlix(React Client) app was built as part of my web development coursework through CareerFoundry. The goal of the project was to create a fun, user-friendly design, where users could login and view their favorite movies. The combined project uses the MERN(MongoDB, Express, React, and Node.js) stack.
